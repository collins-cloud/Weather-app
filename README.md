# 🌤️ Weather App

A simple and responsive weather application that allows users to search for any city and get real-time weather information.

---

# 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays temperature in Celsius
- 💧 Shows humidity level
- 🌬️ Displays wind speed
- 🌤️ Dynamic weather icons
- ❌ Error message for invalid city names

---

# 🛠️ Built With

- HTML
- CSS
- JavaScript
- OpenWeather API

# ⚙️ How It Works

1. User enters a city name
2. App sends request to OpenWeather API
3. Weather data is fetched using "fetch()"
4. Data is displayed on the UI
